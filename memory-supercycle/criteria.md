# Scoring Criteria — Memory Supercycle Edition

Total score out of 100, weighted across 5 dimensions plus signal adjustments.

## 1. Memory Chokepoint Strength (30 points)

This is the primary dimension. Score using the 5-question memory chokepoint test from `program.md`.

| Chokepoint Score (out of 25) | Points | Tier |
|------------------------------|--------|------|
| 21-25 | 26-30 | **Monopoly / Duopoly Chokepoint** |
| 16-20 | 20-25 | **Tight Oligopoly with Moat** |
| 11-15 | 14-19 | **Qualified Supplier with Leverage** |
| 6-10  | 7-13  | **Cycle Rider** |
| 1-5   | 1-6   | **Commodity Exposure** |

### What gets a high score

- Only 1-3 qualified vendors
- Product is needed for AI systems to ship
- Replacement takes quarters or years
- Capacity expansion depends on tools, yields, and cleanrooms

## 2. Financial Strength (25 points)

| Score | Criteria | Memory Signal |
|-------|----------|---------------|
| 21-25 | High margins, positive FCF, strong balance sheet | Structural scarcity is showing up in the P&L |
| 16-20 | Solid profitability, healthy balance sheet | Good cycle leverage with discipline |
| 11-15 | Improving but uneven margins or FCF | Recovery is real, but execution still matters |
| 6-10  | Weak margins, negative FCF, leverage concerns | Company may be on the wrong side of the cycle |
| 1-5   | Cash burn or poor balance sheet | Avoid unless the choke is overwhelming |

For memory makers, gross and operating margins matter.
For equipment / test names, operating margin, FCF, and book-to-bill visibility matter.

## 3. AI / Memory Mix Leverage (20 points)

| Score | Criteria |
|-------|----------|
| 17-20 | Revenue increasingly driven by HBM, server DRAM, AI SSD, or HBM packaging / test |
| 13-16 | Meaningful AI memory exposure, but still mixed with conventional demand |
| 9-12  | Moderate AI exposure, broader semiconductor recovery needed |
| 5-8   | Mostly indirect cycle exposure |
| 1-4   | Very little direct AI memory leverage |

The best names have direct exposure to:
- HBM
- Server DDR5 / LP memory
- Enterprise SSD for inference
- HBM bond / probe / test intensity

## 4. Capacity Advantage & Supply Discipline (15 points)

| Score | Criteria |
|-------|----------|
| 13-15 | Has scarce capacity, strong roadmap, and can expand without destroying returns |
| 10-12 | Good roadmap and reasonable capex posture |
| 7-9   | Can benefit, but supply additions or execution are uncertain |
| 4-6   | Capacity may lag or overspend may kill returns |
| 1-3   | No real capacity edge |

Key things to watch:
- 1-gamma / 1c / next-gen node ramps
- HBM4 and packaging capacity
- Tester and probe capacity
- Enterprise SSD output tied to AI demand

## 5. Timing & Valuation (10 points)

| Score | Criteria |
|-------|----------|
| 9-10  | Strong cycle signal, attractive valuation, supportive technicals |
| 7-8   | Good setup with clear near-term catalyst |
| 5-6   | Fairly valued and technically neutral |
| 3-4   | Strong thesis but priced aggressively |
| 1-2   | Great story, poor entry |

## Signal Adjustments

### Bullish
| Signal | Adjustment |
|--------|------------|
| Capacity or supply sold out into next year | +3 |
| First-source or lead qualification for HBM4 / next-gen memory | +3 |
| Enterprise SSD / server DRAM demand called out as AI-driven | +2 |
| New cleanroom / packaging / tester capacity with firm demand | +2 |
| RSI < 35 and thesis intact | +2 |

### Bearish
| Signal | Adjustment |
|--------|------------|
| Major customer concentration >50% | -3 |
| Delayed HBM qualification or competitive slippage | -3 |
| Consumer / foundry drag overwhelms memory upside | -2 |
| RSI > 70 | -2 |
| Geopolitical manufacturing concentration | -3 |
| Evidence of oversupply or reckless capex | -3 |

- Are test / packaging names offering better risk-adjusted returns than memory fabs?
- Is the market paying too much for obvious AI winners?
